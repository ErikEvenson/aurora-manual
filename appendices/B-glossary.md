# Appendix B: Glossary

This glossary defines key terms used throughout the Aurora C# manual. Cross-references to detailed coverage are provided for major topics.

| Term | Definition |
|------|-----------|
| Accessibility | A value from 0.1 to 1.0 indicating how easily a mineral deposit can be mined; higher values mean more output per mine |
| Active Sensor | A sensor that emits energy to detect targets; provides range and resolution data but reveals the emitting ship's position via EM signature |
| AMM | Anti-Missile Missile; a small, fast missile designed to intercept incoming enemy missiles |
| Armor Column | One vertical segment of a ship's armor grid; damage is applied to random columns, requiring multiple hits on the same column to penetrate |
| ASM | Anti-Ship Missile; a missile designed to damage or destroy enemy vessels, as opposed to AMMs |
| Beam Fire Control (BFC) | A fire control component that directs beam weapons at targets; determines tracking speed, range, and ECCM level |
| Boost | A modifier applied to engines that increases power output at the cost of exponentially higher fuel consumption |
| BP | Build Points; the unit of production capacity used to construct installations, ships, and components |
| Bridge | A required component on any ship intended to leave its home system; without one, the ship cannot operate independently |
| CC | Colony Cost; a measure of how hostile a planet is to colonization, determining infrastructure requirements |
| CIWS | Close-In Weapon System; a combined sensor and gauss cannon that automatically fires at incoming missiles at point-blank range |
| Cloak | A device that reduces a ship's thermal and EM signatures by a percentage, making it harder to detect |
| Colony Cost | See CC |
| Commander | A named officer assigned to lead a fleet, colony, or sector; provides skill bonuses to operations |
| Commercial Engine | An engine type that is larger but more fuel-efficient, designed for freighters and civilian vessels |
| Conventional Industry | Basic industrial capacity that does not require TN minerals; produces at 1/10th the rate of TN construction factories |
| Conventional Start | A game start option where the player begins with pre-TN technology and must research basic Trans-Newtonian physics |
| Crew Grade | A measure of crew training level affecting ship performance, maintenance, and morale |
| DAC | Diplomat Assigned to Contact; affects diplomatic relations with NPRs |
| Damage Control | The process by which engineering crew repair damaged components during and after combat |
| DP | Deployment Points; fuel harvesters and other automated craft have deployment limits |
| ECM | Electronic Countermeasures; a system that reduces enemy fire control accuracy, each level providing 10% reduction |
| ECCM | Electronic Counter-Countermeasures; mounted on fire controls, each level negates one level of target ECM |
| EM | Electromagnetic; one of the two passive sensor detection types, detecting active sensor emissions and shield signatures |
| EM Signature | The electromagnetic emission profile of a ship, generated by active sensors, shields, and certain components when operating |
| Engine Power | The thrust output of an engine, measured in EP; divided by ship tonnage to determine speed |
| EP | Engine Power; the raw thrust output of an engine component |
| Estimated Thermal Signature | What enemy sensors see when scanning your ship; affected by engine power and thermal reduction technology |
| FC | Fire Control; a component required to direct weapons at targets, available in beam and missile variants |
| Fire Delay | The time between shots for a beam weapon; affected by weapon type and rate of fire technology |
| Fleet | A group of one or more ships operating under unified orders and a single commander |
| Fuel Harvester | An installation or ship module that collects Sorium from gas giant atmospheres |
| Fuel Refinery | An installation that converts Sorium into usable fuel for ships |
| Gauss Cannon | A rapid-fire kinetic weapon dealing 1 damage per shot; excellent for point defense due to high fire rate |
| Geosurvey | Geological Survey; the process of scanning a planetary body to discover mineral deposits |
| Governor | A commander assigned to manage a colony, providing production and growth bonuses |
| Gravitational Survey | The process of exploring a system to discover jump points leading to other star systems |
| Ground Forces | Military units designed for planetary assault and defense, distinct from naval forces |
| HS | Hull Size; 50 tons of ship mass, the basic unit of ship component sizing (1 HS = 50 tons) |
| HTK | Hit to Kill; the number of damage points a component can absorb before being destroyed |
| Infrastructure | An installation required to support population on planets with colony costs above 0 |
| JP | Jump Point; a naturally occurring or artificially created portal between star systems |
| Jump Drive | A ship component that enables transit through jump points; available in military (self-jump) and commercial (squadron transit) variants |
| Jump Gate | A permanent structure built at a jump point that allows any ship to transit without a jump drive |
| Laser | A beam weapon with good range and consistent damage; the most versatile beam weapon type |
| Magazine | A ship component that stores missiles and feeds them to launchers; can explode catastrophically if hit while loaded |
| Maintenance Clock | A timer on each ship indicating time since last overhaul; when exceeded, components have increasing failure chance |
| Maintenance Facility | An orbital installation that overhauls ships, resetting their maintenance clock |
| Meson Cannon | A beam weapon that bypasses both shields and armor, striking internal components directly; short range and low accuracy |
| Microwave Focal Size | Determines the range bracket of a laser; larger focal sizes give longer range but larger component size |
| Military Engine | A standard engine type optimized for power output; smaller than commercial engines but less fuel-efficient per EP |
| Mineral | One of eleven Trans-Newtonian elements required for advanced construction: Duranium, Neutronium, Corbomite, Tritanium, Boronide, Mercassium, Vendarite, Sorium, Uridium, Corundium, Gallicite |
| Missile Fire Control (MFC) | A fire control that guides missiles to their targets; determines maximum engagement range and ECM/ECCM ratings |
| Missile Launcher | A ship component that fires missiles; reload time depends on launcher size and technology |
| MSP | Maintenance Supply Points; consumable supplies used for shipboard repairs and maintenance |
| Multi-stage Missile | A missile design using a booster stage for long range and a separating attack stage for terminal engagement |
| NPR | Non-Player Race; a conventional alien civilization generated by the game with its own technology, fleets, and diplomacy |
| Ordnance | General term for missiles and other expendable weapons carried by ships |
| Overhaul | The process of resetting a ship's maintenance clock at a maintenance facility; required periodically to prevent breakdowns |
| Particle Beam | A beam weapon with shorter range than lasers but no damage falloff and good armor penetration |
| Passive Sensor | A sensor that detects the thermal or EM emissions of other ships without revealing the detecting ship's presence |
| PD | Point Defense; weapons or modes designated to intercept incoming missiles and projectiles |
| Plasma Carronade | A short-range beam weapon dealing triple damage; devastating at close range but useless at distance |
| Power Plant | A ship component that generates power for beam weapons; required tonnage depends on total beam weapon power draw |
| Precursor | A spoiler race representing an ancient, technologically advanced civilization with powerful static defenses |
| Railgun | A kinetic beam weapon that ignores a number of armor layers equal to its damage rating |
| Refueling Hub | A fleet order configuration allowing tankers to automatically refuel other ships in the fleet |
| Resolution | For sensors, determines what size target the sensor is optimized to detect; for fire controls, affects range |
| RP | Research Points; the unit of scientific progress, generated by research labs and scientists |
| Ruins | Alien archaeological sites on planets that can be explored for technology bonuses and resources |
| Sector | An administrative grouping of colonies under a sector commander, providing additional production bonuses |
| Shield | A defensive system that absorbs incoming damage before it reaches armor; regenerates over time but generates EM signature |
| Shipyard | An orbital facility that constructs, refits, and repairs ships; has tonnage limits and retooling requirements |
| Slipway | An individual construction bay within a shipyard; each slipway can build one ship at a time |
| Sorium | A Trans-Newtonian mineral used as fuel for ship engines (after refining) and fuel harvester feedstock |
| Spoiler Race | A pre-designed alien threat with unique mechanics and no diplomacy options, serving as a major game challenge |
| Spinal Mount | A weapon mounting that allows a beam weapon to be built larger (up to hull size), gaining damage at the cost of limited firing arc |
| Strikegroup | A task group sub-formation used for organizing fleet elements |
| Survey Point | A location in a system that must be checked during gravitational survey to find jump points |
| Swarm | A spoiler race that overwhelms through sheer numbers and regenerates forces over time |
| Task Force | A higher-level organizational unit containing multiple task groups |
| Task Group | The basic organizational unit of ships in Aurora; all orders are given to task groups |
| Terraforming | The process of modifying a planet's atmosphere to reduce colony cost; uses terraforming installations |
| Thermal Signature | The heat emission profile of a ship, primarily generated by engines; detectable by thermal passive sensors |
| TN | Trans-Newtonian; refers to the advanced physics and technology that enables interstellar civilization in Aurora's setting |
| TN Start | A game start option where the player begins with basic Trans-Newtonian technology already researched |
| Tracking Speed | The maximum target speed a fire control or weapon can accurately engage; targets faster than this suffer reduced hit chance |
| Turret | A mounting that increases a beam weapon's tracking speed (4x for single weapon, 3x per weapon for multi-weapon), at the cost of increased size |
| Wealth | The economic currency used to fund installations, ship maintenance, and other expenses |
| Waypoint | A designated point in space used for fleet navigation orders |
| Xenoarchaeology | The study of alien ruins, conducted by xenoarchaeology teams to recover technologies and artifacts |
