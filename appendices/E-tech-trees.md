# Appendix E: Technology Trees

*Added: v2026.01.25*

This appendix provides visual technology tree diagrams for all research categories in Aurora C#.
Technologies are color-coded by research cost:

| Color | Cost Range | Game Phase |
|-------|------------|------------|
| Green | 0-500 RP | Conventional/Starting |
| Blue | 500-10K RP | Early Trans-Newtonian |
| Yellow | 10K-100K RP | Mid Game |
| Purple | 100K-1M RP | Late Game |
| Pink | 1M+ RP | Endgame |

External prerequisites (technologies from other categories) are listed below each diagram.
