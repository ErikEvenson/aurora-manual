## Execution Strategy

### Phase 1: Charts (Waves 1-5)
Run 5 parallel agents, each generating ~8 charts using matplotlib.
Charts require:
- Reading source data from manual sections
- Creating appropriate visualization (bar, line, scatter, matrix)
- Saving to images/charts/ directory
- Updating markdown files with image references

### Phase 2: Diagrams (Waves 6-12)
Run 7 parallel agents, each generating ~8-12 diagrams.
Diagrams require:
- Reading workflow/process descriptions from manual
- Creating flowcharts, decision trees, matrices
- Saving to images/diagrams/ directory
- Updating markdown files with image references

### Phase 3: Screenshots (Wave 13)
Create tracking issues for manual capture requirements.
Screenshots need:
- Actual Aurora C# game running
- Specific game state/window visible
- Annotation with callouts
- Manual upload and integration

- Charts: `images/charts/{section}-{name}.png`
- Diagrams: `images/diagrams/{section}-{name}.png`
- Screenshots: `images/screenshots/{chapter}/{section}-{name}.png`
